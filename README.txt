Данное тестовое задание было выполнено с использованием WebAPI. Для тестирования функционала API необходимо обратиться по адресу https://localhost:7146. Для удобства использования предоставлены два способа: можно воспользоваться Postman с использованием приложенного json файла(FilterToDelivery.postman_collection.json), либо воспользоваться Swagger, который также настроен на адрес https://localhost:7146/swagger.
В качестве СУБД была использована SQLite

Был добавлен небольшой "crud" для возможности добавления и удаления заказов. Добавлены внутренние логи.

Соответственно для фильтрации реализован get запрос с двумя входящими параметрами (CityDistrict,FirstDeliveryDateTime)

Также результат фильтрации заказов записывается в файл data.txt. В случае необходимости изменения адреса API можно отредактировать файл launchSettings.json.
